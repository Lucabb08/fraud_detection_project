# Project Proposal

The aim of this project is to compare several classification methods in an unbalanced dataset context in order to detect whether a payment is fraudulent.

Several methods will be tested:
- Logistic Regression
- KNN
- Decision Tree
- Random Forest
- AdaBoost
- Bagging
- Ensemble Voting

With several evaluation metrics in order to compare them:
- AUPRC
- Recall
- Precision
- F1 Score
